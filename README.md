# Nkwa Bible

Nkwa Bible is a beautifully designed offline Bible app that helps Christians read, study, and meditate on God's Word.

## Features

- Twi and English Bible
- Offline Reading
- Daily Bible Verses
- Verse Search
- Bookmarks
- Personal Notes
- Continue Reading
- Simple and Modern Interface
